# Local_Food_Wastage_Management_System

A robust streamlit app that can help manage food wastage by allocating remaining food from different sources of good quality to the individual/organisation in need. This ensures huge reduction in food wastage and helps in meeting the food requiremnts who are in need.

This app is designed with streamlit, SQLite for database management, and Plotly for data visualization.

This app also has a SQL Playground section that can be used to run custom SQL Queries for Analysis.
